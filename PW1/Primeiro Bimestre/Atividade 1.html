HTML

HTML é uma linguagem para estruturação e apresentação de conteúdo para o World Wide Web

HTML 5 é a mais recente evolução do padrão que define a linguagem HTML

Esta versão traz importantes mudanças quanto ao papel do HTML no mundo da Web, atráves de novas funcionalidades como sêmantica e acessibilidade

Possibilita o uso de novos recursos antes possíveis apenas com a aplicação de outras tecnologias
Muitos recursos do HTML 5 têm sido construídos coma intenção de serem executados em dispositivos móveis como smarthones e tablets

História

Entre 1991 e 1999, o HTML se deselvolveu a partir da versão 1 até a versão 4. Em 2008, a primeira versão públicada do HTML 5 foi lançada.

As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familaridade sem familiares com a Web. Com o tempo aumentou a tendência em tornar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web desenvolvidas com as primeiras versões do HTML.
