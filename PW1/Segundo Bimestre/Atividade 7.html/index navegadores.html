Alguns Navegadores associadas ao Desenvovimento Web
 Firefox Mozilla Firefox é um navegador livre e multi-plataforma desenvolvido pela Mozilla Foundation com ajuda de centenas de colaboradores. A intenção da fundação é desenvolver um navegador leve, seguro, intuitivo e altainente extensivel
 Chrome O Google Chrome é um navegador de internet, desenvolvido pela companhia Google com visual minimalista.
 Internet  Explorer Internet Explorer é uma série de navegadores web gráficos desenvolvidos pela Microsoft e inclusos como parte da linha de sistemas operacionals Microsoft Windows, iniciado em 1995
 Safari O Safari é um navegador desenvolvido pela Apple Inc. e incluido como o navegador padrão a partir do sistema operacional Mac OS X v10.3 (Panther). Apresenta uma interface simples, característica dos produtos da Apple.
 Netscape Netscape Navigator é um navegador web proprietário que foi o principal produto da Netscape Communications Corporation até 2002.
